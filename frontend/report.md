学校心理测评统计分析报告（结构整理）

一、整体导出思路

1. 主报告

给校领导看，重点是：

- 全校整体情况

- 各年级对比

- 各班级对比

- 重点风险分布

- 需要优先关注的对象数量和范围

- 管理建议

2. 学生附录

给心理老师、德育主任、校领导备查，重点是：

- 每个学生的测评结果摘要

- 风险等级

- 维度得分

- 重点关注维度

- 建议措施

这样做的好处是：

- 领导首页就能看到关键结论

- 不把几十上百个学生明细堆在前面

- 正式、层级清晰、可打印归档

二、建议分成 6 个模块

模块 1：封面与报告说明

这是领导第一眼看到的内容，建议单独 1 页。

内容建议

- 报告标题：学校心理测评统计分析报告

- 学校名称

- 测评时间范围

- 报告生成时间

- 数据范围说明：
  - 覆盖年级数量

  - 覆盖班级数量

  - 总学生数

  - 已测评人数

  - 未测评人数

- 统计口径说明：
  - 风险等级定义（R0 / R1 / R2 / R3）

  - 本报告仅用于校内心理健康筛查与教育管理参考

  - 不作为医学诊断依据

页面风格建议

封面可以比学生报告更正式一点：

- 学校 logo / 校名

- 简洁蓝灰色风格

- 大标题 + 副标题

- 页脚标注“内部资料，请注意保密”

模块 2：全校总体概览（领导最关心）

这一部分建议放在正文第一页，是整份 PDF 最核心页面。

展示内容

1）测评完成情况

- 总学生数

- 已完成测评人数

- 未完成测评人数

- 测评完成率

2）全校风险等级分布

- R0 低风险人数 / 占比

- R1 中低风险人数 / 占比

- R2 中高风险人数 / 占比

- R3 高风险人数 / 占比

3）全校维度均值概览

展示主要心理维度平均值，例如：

- 情绪状态

- 人际关系

- 学习压力

- 家庭支持

- 自我认知

- 行为适应

可用：

- 雷达图

- 或者横向条形图

4）关键提示

自动生成 3~5 条领导摘要，例如：

- 当前全校测评完成率为 92.6%，整体覆盖较高。

- 高风险学生占比 4.3%，主要集中在八年级和高一年级。

- 学习压力、人际关系为当前全校波动较明显的重点维度。

- 建议优先安排高风险学生复核与班主任联动关注。

为什么要有这一页

领导通常先看：

- 有没有做完

- 风险高不高

- 哪些年级最突出

- 现阶段要不要干预

所以这一页必须是 “摘要式决策页面”。

模块 3：年级统计分析

这一部分是从学校整体下钻到年级层级。

展示内容

建议每个年级显示一行，或一页做完整分析表。

1）年级风险分布表

字段建议：

年级

学生数

已测评

完成率

R0

R1

R2

R3

平均风险分

2）年级对比图

推荐图表：

- 堆叠柱状图：各年级 R0~R3 分布

- 折线图：各年级平均风险分

- 热力表：各年级主要维度均值

3）年级结论摘要

自动生成文字，例如：

- 七年级整体稳定，低风险占比较高。

- 八年级中高风险比例明显高于其他年级，需重点关注。

- 高一年级学习压力维度得分偏高，建议结合阶段性考试压力开展团辅。

这一模块的价值

让领导明确：

- 哪个年级是重点

- 是否与毕业班、升学阶段有关

- 资源应该先投到哪里

模块 4：班级统计分析

这一部分继续下钻到班级，是班主任管理和德育管理最有用的部分。

1）班级统计总表

- 预警人数 = R2 + R3

- 可以按预警人数倒序排序

2）重点班级排名

建议单独列一个 TOP 表：

3）班级图表

推荐：

- 班级堆叠柱状图

- 风险人数 TOP10 班级柱状图

页面建议

因为班级多，这里不要每个班做太重的图。

建议：

- 先放 全班级总表

- 再放 TOP 风险班级图

- 最后给出简要结论

给领导看的总结话术

- 中高风险学生主要集中在 X 年级 X 班、X 年级 X 班。

- 个别班级测评完成率偏低，建议补测。

- 建议对预警人数较高班级开展班级团体辅导及班主任协同观察。

模块 5：重点学生预警摘要（领导版）

这里很重要，但不建议把所有学生详细报告都堆进正文。

正文里只放 重点学生摘要清单。

展示内容

建议只放：

- R3 全部学生

- R2 中风险学生

- 或者“预警名单前 N 人”

字段建议：

序号

姓名

年级

班级

风险等级

风险分

主要预警维度

测评时间

重点提示

这页一定要加：

- 保密说明

- 仅限校内授权人员查阅

展示原则

领导版正文只展示：

- 必要识别信息

- 风险等级

- 关注维度

- 不展示过多隐私描述

这样既满足管理需要，也更稳妥。

模块 6：学生测评结果附录（全量）

你要求“包括每个学生的心理统计结果”，建议把它放在 附录 中。

这部分可以按：

- 年级

- 班级

- 学号 / 姓名

排序

三、学生附录推荐格式

附录每个学生建议占 半页到一页，不要像弹窗那样完全截图搬进去，而是做成统一结构化卡片。

每个学生建议包含以下内容

1）基本信息

- 姓名

- 年级

- 班级

- 测评状态

- 测评时间

2）结果摘要

- 风险等级

- 风险分

- 综合结论

- 是否建议复核 / 转介

3）维度统计

可做成小表格：

维度

得分

状态

情绪状态

72

需关注

81

较高

58

正常

家庭支持

64

轻度关注

4）重点关注维度

例如：

- 学习压力

- 情绪波动

- 同伴关系

5）建议摘要

分三类最合适：

- 给学生的建议

- 给教师的建议

- 给家长的建议

6）备注

- 复评建议周期

- 建议跟进动作

四、页码结构建议

建议 PDF 最终结构如下：

A. 封面 1 页

B. 目录（可选） 1 页

C. 主报告

- 第 1 页：全校总体概览

- 第 2 页：年级统计分析

- 第 3 页：班级统计分析

- 第 4 页：重点学生预警摘要

- 第 5 页：管理建议与后续工作建议

D. 附录

- 按年级/班级输出全部学生结果 可能很多页

五、给领导看的“管理建议模块”建议单独加一页

虽然上面分 6 个模块已经够了，但在正式场景里，我建议把“建议”做成主报告最后一页，这样更完整。

建议内容

1）学校层面

- 对高风险学生建立跟踪台账

- 建立月度复评机制

- 对重点年级开展专题心理活动

- 优化补测机制，提升覆盖率

2）年级层面

- 对高波动年级安排定向心理讲座

- 班主任加强阶段性观察

3）班级层面

- 重点班级开展团辅

- 对预警学生进行个案访谈

4）家校协同

- 适度开展家长沟通

- 对需要持续关注学生建立家校联动机制

六、格式上建议“主报告”和“附录”区分明显

主报告风格

- 更正式

- 统计图 + 表格 + 摘要结论

- 偏“汇报材料”

学生附录风格

- 更结构化

- 一人一块

- 偏“档案材料”
