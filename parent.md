# 家长测评

# 一、评分规则设计

## 1 基础评分

每题选项：

| 选项 | 分值 |
| ---- | ---- |
| 从不 | 0    |
| 很少 | 1    |
| 有时 | 2    |
| 经常 | 3    |
| 总是 | 4    |

---

## 2 反向题处理

数据库已有字段：

```
reverse_scored
```

如果为 `1`，需要反向：

```
score = 4 - score
```

例如：

| 选择 | 原始 | 反向 |
| ---- | ---- | ---- |
| 总是 | 4    | 0    |
| 经常 | 3    | 1    |
| 有时 | 2    | 2    |
| 很少 | 1    | 3    |
| 从不 | 0    | 4    |

---

# 二、维度评分计算

每个 **domain 100题**

```
domain_score = sum(question_scores)
```

理论范围：

```
0 - 400
```

转换为百分制：

```
domain_index = (domain_score / 400) * 100
```

最终得到：

```
0 - 100
```

# 六、教养方式分类（关键）

根据 3 个指标分类：

```
parenting_style
discipline_method
emotional_support
```

生成 4 种类型：

| 类型   | 特征              |
| ------ | ----------------- |
| 权威型 | 高支持 + 合理规则 |
| 专制型 | 高控制 + 低沟通   |
| 放任型 | 高支持 + 低规则   |
| 忽视型 | 低支持 + 低参与   |

判断逻辑示例：

```
if support > 60 and discipline > 60
 -> 权威型

if discipline > 70 and support < 40
 -> 专制型

if support > 60 and discipline < 40
 -> 放任型

if support < 40 and discipline < 40
 -> 忽视型
```
